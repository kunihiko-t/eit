gulp-boilerplate
====

Requirements
---

 * NodeJS

 * Gulp

 * Ruby

 * Compass

Getting started
---

```bash
git clone git@bitbucket.org:tatsuo/gulp-boilerplate.git
cd gulp-boilerplate

npm install

bower install

gulp

# or

gulp --release # this enables compression
```


Add vendor scripts
---

```bash
bower install jquery --save
```
